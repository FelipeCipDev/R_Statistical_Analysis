# Projeto 4: Análise Estatística Avançada com R

Este projeto demonstra análises estatísticas avançadas utilizando R, incluindo testes de hipóteses, séries temporais, regressão e análise de sobrevivência. Ideal para portfólio, estudos e demonstração de habilidades analíticas.

## Estrutura

- `Análises_Estatísticas/`: Scripts R para cada tipo de análise
- `Relatórios/`: Modelos de relatório (RMarkdown), dashboards (Shiny) e apresentações
- `Datasets/`: Exemplos de dados financeiros, saúde e marketing

## Como Executar

1. Abra os scripts `.R` no RStudio ou outro ambiente R
2. Execute cada script para ver exemplos de análise com dados simulados
3. Para gerar um relatório, abra o arquivo `.Rmd` em `Relatórios/R_Markdown_Reports` e clique em "Knit"

## Exemplos de Análises

- **Testes de Hipóteses**: t-teste, ANOVA
- **Séries Temporais**: decomposição, visualização
- **Regressão Avançada**: regressão logística (GLM)
- **Sobrevivência**: Kaplan-Meier, análise de tempo até evento

## Dados de Exemplo
